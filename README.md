## 📊 Project Overview

This project evaluates the effectiveness of a **digital loyalty program** using statistical analysis in R.

The analysis compares:
- Existing vs New customers  
- Spending before vs after the program  

---

## 🎯 Objectives

- Determine if spending differs between customer types  
- Measure impact of loyalty program  
- Apply statistical testing  
- Validate results using Copilot  

---

## 📈 Key Analysis

### 🔹 Independent Samples t-test

**Purpose:** Compare spending after program between customer types  

- t = 0.318  
- df = 48  
- p-value = 0.752  

👉 Conclusion: No significant difference between groups :contentReference[oaicite:0]{index=0}  

---

### 🔹 Paired Samples t-test

**Purpose:** Compare spending before vs after program  

- t = 2.887  
- df = 49  
- p-value = 0.0058  

👉 Conclusion: Loyalty program significantly changed spending :contentReference[oaicite:1]{index=1}
